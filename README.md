# 🪙 CoinTC Miner

CoinTC Miner is a Telegram Mini App built with HTML, CSS and JavaScript.

Mine CoinTC by tapping the coin, earn points, manage your energy and enjoy a smooth Telegram WebApp experience.

---

## ✨ Features

- ⚡ Tap to Mine
- 🔋 Energy System
- 💾 Local Storage Save
- 📱 Telegram Mini App Ready
- 🎮 Responsive Mobile UI
- ✨ Haptic Feedback
- 🚀 Fast & Lightweight

---

## 🛠️ Technologies

- HTML5
- CSS3
- JavaScript
- Telegram WebApp API

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/hzrtiman9-gif/CoinTC.git
```

Open the project

```bash
cd CoinTC
```

Deploy using GitHub Pages, Netlify or Vercel.

---

## 🎮 Gameplay

- Tap the CoinTC coin.
- Earn CoinTC.
- Every tap consumes energy.
- Energy regenerates automatically.
- Progress is saved locally.
